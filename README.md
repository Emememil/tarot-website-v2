## Technical Deep Dive

### 1. Hybrid Architecture (Web + Systems)
My work bridges the gap between high-level UI and low-level performance.
- **Cinera App:** Explores the high-performance world of **Tauri and Rust**, bringing a sleek React frontend to a native desktop environment.
- **Spotify Dashboard:** Implements complex data visualization and third-party API orchestration.
- **Expense Splitter Pro:** A PWA focusing on robust state management and real-time UI updates.

### 2. UI/UX Architecture
- **Reusable Animation Wrappers:** Built a custom `<AnimatedContent />` component leveraging `framer-motion` to handle `whileInView` logic, custom delays, and load-triggers globally.
- **Accessible Interactions:** Modal systems feature **body-scroll locking** and **keyboard event listeners (Esc)** for a native-app feel.
- **Clean Component Logic:** Separation of concerns between data (`projects.js`), styling (CSS Modules), and presentation logic.
